几个快捷命令

Posted on 2021-10-25

讲几个比较好用的命令：

一个是curl。我们经常搭建好项目后写一个测试用的接口，让后浏览器直接访问路径，打印一个hello world。我想基本大家都是这么干的。实际上像这种简单测试，用黑窗口的curl命令就可以代替，无论是linux或者windows都是可以使用。

比如我的springboot项目搭建好了，测试接口路径是 /hello ，项目名my_project。默认808端口。

以前我会打开chrome，输入http://localhost:8080/my_project/hello ,然后屏幕打印一句话

这个就算是通了。

现在我们可以直接打开黑窗口

输入：curl http://localhost:8080/my_project/hello

就会同样的在下一行打印

另一个是tree命令，这个windows自带。如果是linux里面需要使用 yum install tree 安装一下

在某个文件夹输入tree命令后，就会在窗口里树状展示整个文件夹的树状图（可能会比较长）

在linux中，tree命令还有许多参数 ，比如tree -h 等，这里不多做介绍
